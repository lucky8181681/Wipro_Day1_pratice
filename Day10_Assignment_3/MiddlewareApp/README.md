# MiddlewareApp

## Objective
Demonstrate ASP.NET Core middleware pipeline and static file handling.

## Features
- Custom middleware for request and response logging
- Global exception handling
- HTTPS redirection
- Static file serving using wwwroot
- Basic Content Security Policy (CSP)

## Steps Performed
1. Created ASP.NET Core Empty Web App
2. Implemented middleware:
   - Logging (request/response)
   - Exception handling
   - HTTPS redirection
3. Enabled static files using wwwroot
4. Added HTML, CSS, and JS files
5. Configured security headers (CSP)
6. Tested application in browser

## Output
Application logs requests in console and serves static web content successfully.
