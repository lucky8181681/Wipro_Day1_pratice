# RazorPagesApp

## Objective
Build a dynamic web application using Razor Pages and PageModel.

## Features
- Razor Pages with PageModel
- Display list of items
- Add new items using form
- Shared in-memory data across pages

## Steps Performed
1. Created ASP.NET Core Web App (Razor Pages)
2. Added pages:
   - Items (display list)
   - AddItem (add new item)
3. Implemented PageModel logic
4. Used static list to share data
5. Connected AddItem and Items pages
6. Tested full flow (add and view items)

## Output
User can add items and view updated list dynamically in the application.
